Deployment Instructions

Local Development

From code/library-api run:

./run.sh

This script:

  1. Creates .env and generates the app key (if needed)
  2. Installs Composer dependencies
  3. Runs migrations and seeds the database
  4. Starts the Laravel API on port 8080

Docker Deployment

From code/library-api run:

./setup.sh

This will:

  1. Build the Docker image
  2. Start the Laravel app and MySQL containers
  3. Run migrations and seed automatically
